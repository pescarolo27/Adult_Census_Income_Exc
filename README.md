# Adult Census Income (Excel)

**The Data:** The dataset is a collection of information about individuals living in the United States, extracted from the 1994 Census database ([Kaggle](https://www.kaggle.com/datasets/uciml/adult-census-income?resource=download)). It contains various demographic, social and economic attributes about each individual. In the Excel file, there are various sheets containing notes about the dataset, the original dataset, a modified version of the data, analyses, & a final dashboard.

**Purpose:** An Excel dashboard was created to display relevant metrics according to individuals' income group.
